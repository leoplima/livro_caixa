## 🎨 Personalizações Fáceis

### Mudar Cores

Abra `config.py` e procure por `COLORS`. Você pode mudar:
```python
COLORS = {
    "primary": "#NOVA_COR",
    # ...
}
```

### Adicionar Categoria

Abra `config.py` e procure por `CATEGORIAS_RECEITA` ou `CATEGORIAS_DESPESA`:
```python
CATEGORIAS_RECEITA = [
    "Sua Nova Categoria",  # ← Adicione aqui
    # ...
]
```

## 🛠️ Solução de Problemas

### A porta 8501 está em uso?

```bash
streamlit run app.py --server.port 8502
```

### Esqueceu a senha do admin?

Abra um terminal e execute:
```bash
del livro_caixa.db
python init.py
```

Isso vai recriar o banco e a senha volta a ser: `Admin123456`

### Erro: "Módulo não encontrado"?

```bash
pip install -r requirements.txt
```

## 📧 Configurar Envio de Emails (Opcional)

Se quiser que os convites sejam enviados por email:

1. Abra `config.py`
2. Procure por `SMTP_SERVER`
3. Configure com seus dados de email:
   ```python
   SENDER_EMAIL = "seu_email@gmail.com"
   SENDER_PASSWORD = "sua_senha_de_app"
   ```

## 💾 Fazer Backup

Para fazer backup dos dados:

### Opção 1: Clique em um arquivo
Clique duas vezes em:
```
fazer_backup.bat
```

### Opção 2: Manual
Copie o arquivo `livro_caixa.db` para um local seguro regularmente.

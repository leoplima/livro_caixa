## 🏛️ Adicionar Marca D'água

### Como Fazer:

1. Fazer login como **admin@igreja.com**
2. Menu → **🏛️ Igrejas**
3. Clique em **➕ Nova Igreja**
4. Preencha os dados
5. Faça upload da imagem (logo da sua igreja)
6. Clique **✅ Criar Igreja**

### Resultado:
- A imagem aparecerá como fundo semi-transparente
- Muda automaticamente ao trocar de igreja
- Fica elegante sem prejudicar a leitura

## 👥 Adicionar Novo Usuário

### Como Fazer:

1. Login como **admin@igreja.com**
2. Menu → **👥 Usuários**
3. Aba **📧 Convidar**
4. Preencha:
   - Email: `novo@usuario.com`
   - Nome: `João Silva`
   - Nível: `Editor` (pode criar transações)
5. Clique **📧 Enviar Convite**

### Resultado:
- Usuário recebe credenciais
- Pode acessar apenas as igrejas atribuídas
- Nível de acesso controlado pelo admin

## 🧬 Estrutura de Banco

### NOVO - Tabela: igrejas
```
id, nome, cnpj, endereco, 
telefone, email, logo_path, 
marca_agua_path, cor_primaria
```

### NOVO - Tabela: usuario_igrejas
```
usuario_id, igreja_id, access_level
(controla permissões por igreja)
```

### MODIFICADE - Transações
```
+ igreja_id (qual igreja pertence)
```

# 🎉 RESUMO DO PROJETO - Livro Caixa da Igreja

## ✨ O Que Foi Criado

Um **sistema web completo de gestão financeira para igrejas** com dashboard interativo, controle de usuários, banco de dados persistente e interface moderna.

## 📦 Arquivos do Projeto

### Arquivos de Código
- **app.py** (700+ linhas) - Aplicação Streamlit principal com interface completa
- **db.py** (400+ linhas) - Sistema de banco de dados SQLite
- **auth.py** (300+ linhas) - Autenticação e controle de segurança
- **config.py** (50+ linhas) - Configurações centralizadas
- **init.py** (100+ linhas) - Script de inicialização

### Arquivos de Documentação

### Banco de Dados
- **livro_caixa.db** - Banco SQLite (criado automaticamente)

## 📊 Estrutura do Banco de Dados

### Tabela: usuarios
```
id (PK)          → ID único
email (UNIQUE)   → Email do usuário
nome             → Nome completo
senha_hash       → Senha encriptada (SHA256)
access_level     → Nível: viewer, editor, admin
ativo            → Ativo (1) ou inativo (0)
data_criacao     → Quando foi criado
ultimo_acesso    → Último login
```

### Tabela: transacoes
```
id (PK)          → ID único
data             → Data da transação
descricao        → Descrição
tipo             → receita ou despesa
valor            → Valor em reais
categoria        → Categoria selecionada
usuario_id (FK)  → Quem criou
notas            → Observações opcionais
data_criacao     → Quando foi criada
data_atualizacao → Última alteração
```

### Tabela: auditoria
```
id (PK)          → ID único
usuario_id (FK)  → Quem fez
acao             → CREATE, UPDATE, DELETE, SECURITY
descricao        → Detalhes da ação
tabela_alvo      → Qual tabela foi afetada
id_alvo          → ID do registro afetado
data_hora        → Quando aconteceu
```

### Tabela: configuracoes
```
id (PK)          → ID único
chave (UNIQUE)   → Nome da config
valor            → Valor armazenado
data_atualizacao → Última alteração
```

## 🚀 Como Começar

### 1. Verificar Requisitos
```bash
python --version  # Deve ser 3.8+
```

### 2. Instalar Dependências
```bash
pip install -r requirements.txt
```

### 3. Inicializar o Sistema
```bash
python init.py
```

Isso vai criar:
- Banco de dados SQLite
- Usuário admin: admin@igreja.com / Admin123456
- 10 transações de exemplo

### 4. Iniciar a Aplicação
```bash
streamlit run app.py
```

A aplicação abrirá em http://localhost:8501

### 5. Fazer Login
- Email: `admin@igreja.com`
- Senha: `Admin123456`

### ⚠️ Primeira Coisa a Fazer
Altere a senha do admin em **⚙️ Meu Perfil** → **Alterar Senha**

## 📞 Suporte

### Problemas Comuns

**Erro: Port 8501 already in use**
```bash
streamlit run app.py --server.port 8502
```

**Erro: ModuleNotFoundError**
```bash
pip install -r requirements.txt
```

**Banco de dados corrompido**
```bash
del livro_caixa.db
python init.py
streamlit run app.py
```

**Esqueceu a senha do admin**
```bash
# Delete o banco e reinitialize
del livro_caixa.db
python init.py
```
